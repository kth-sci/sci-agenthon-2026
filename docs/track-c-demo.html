A short detour · Kahneman

Two systems of thinking.

Kahneman: the mind runs on two complementary systems. Both are needed — neither alone is enough.

SYSTEM 1

Fast, intuitive

PATTERN-MATCHING · BLACK-BOX
  • Foundation models — GPT, Claude, Gemini
  • Fluent, confident, fast
  • Can be wrong on novel or edge cases
SYSTEM 2

Slow, deliberate

REASONING · EXPLICIT · TRACEABLE
  • The actual process of doing science
  • Plans, experiments, validates, iterates
  • Catches what intuition gets wrong

The Big Picture

AI agents are not faster autocomplete.

Kahneman's two systems of thinking. Both matter for science. Most people are still using System 1.

System 1 · Fast & Intuitive

Pattern matching

FOUNDATION MODELS · GPT · CLAUDE · GEMINI
  • Generate text, code, images fluently
  • Summarise, translate, reformulate
  • Black-box intuition — no explicit reasoning
  • Confident even when wrong on novel cases
System 2 · Slow & Deliberate

Reasoning agents

AI AGENTS · PLAN · ACT · VALIDATE · ITERATE
  • Hypothesis generation at scale
  • Data analysis: design → code → validate → iterate
  • Instrument control: plans the experiment, then runs it
  • Software cost → effectively zero
The shift: from a tool you prompt to a collaborator that acts. The demos today are System 2 already working in a real research lab.

Demo 04 · Virtual Cell Lab

A sandbox where agents
learn to do science.

A virtual E. coli cell built on the mechanistic whole-cell model from the Covert Lab (Stanford). Agents perturb genes, observe simulated growth, plan the next experiment — and can be graded against the model's ground truth.

Why it matters

Ground-truth sandbox

In the wet lab we cannot tell whether the agent's hypothesis is right in real time. In the virtual cell we can. This is where we prove the autonomous-discovery loop — before scaling to a virtual human cell and to physical instruments.

Methodology proven here → scales to wet lab
How it works

Agent-driven in-silico experiments

The agent perturbs genes via a clean API, reads out simulated phenotypes (growth rate, metabolic flux, morphology), proposes the next perturbation, and iterates — the same loop as the physical REEF lab.

Built with Covert Lab · Stanford · mechanistic WCM
