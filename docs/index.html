What Can AI Agents Do?

Agents go beyond chat — they plan, use tools, and iterate toward a goal. Here are three domains where they excel.

Research

Search the literature, summarise papers, run analysis pipelines, generate figures, and iterate on hypotheses — all with natural language commands.

Teaching

Auto-grade submissions with rubric-aware feedback, build Q&A bots from course slides, generate personalised practice problems, and track student progress.

Productivity

Automate KTH admin workflows, draft grant sections and reports, manage scheduling, and connect any web UI or API with browser-automation agents.
