# Start Here

This is the first document for Track B. In the first 10 minutes, use it to understand the exercise and choose a sub-team.

## Track B in one minute

You will use an AI coding agent on a small research-computing workflow built around a harmonic oscillator. The scientific toy is small; the workflow pattern is the point.

The aim is to practise:

- scoping an agent to a clear task;
- asking for inspect, plan, approve, edit, verify;
- separating verified facts from assumptions;
- producing one artifact that can be regenerated by a command.

## Choose a sub-team

**01 HPC simulation workflow**  
Choose this if you want to work with parameter sweeps, local smoke tests, sample SLURM files, logs, and pre-submission checks.

**02 Research-code debugging**  
Choose this if you want to diagnose a physically suspicious simulation, make a small fix, and add a test-first numerical extension.

**03 Scientific post-processing**  
Choose this if you want to build a reproducible analysis pipeline that supports data interpretation with computed evidence.
