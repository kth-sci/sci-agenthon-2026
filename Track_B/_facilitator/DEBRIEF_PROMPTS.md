# Track B Debrief Prompts

Use one or two at the end of the track or after final demos.

## Prompt 1: Verification Before Trust

```text
What did the agent say that sounded plausible before you had evidence?
What check made it trustworthy or exposed that it was incomplete?
```

## Prompt 2: From Toy to Lab

```text
Where does your own research currently rely on memory, manual steps, or one-off notebooks?
What is the smallest agent-assisted artifact that would make it more reproducible?
```

## Prompt 3: Useful Skepticism

```text
Which optional team role was most important today?
Why?
```

## Prompt 4: The Next Safe Automation

```text
What should be automated next, and what should remain a human decision?
```

## Prompt 5: Teaching Transfer

```text
How could this same pattern help students learn scientific computing without making them verify AI-generated code less carefully?
```
