# SCI Agentathon 2026

This repo contains materials for a one-day agentic AI hackathon at KTH SCI school for faculty and researchers. The event is organised in three tracks, each exploring a different family of agentic problems.

## Repository layout

```
sci-agenthon-2026/
  CLAUDE.md          # this file
  docs/              # GitHub Pages event website (index, program, Track C guide, slides)
  Track_B/           # Track B skeleton and materials (research and HPC workflow agents)
  Track_C/           # Track C skeleton and materials (open agent exploration)
  Agentathon_2026_*.pdf  # official event documents (full details, schedule, contacts)
```

## Tracks

- **Track A** — course syllabus to teaching assistant (not in this repo; separate workspace)
- **Track B** — research and HPC workflow agents; materials in `Track_B/`
- **Track C** — open agent exploration (KTH systems, knowledge work, lab automation); materials in `Track_C/` and `docs/`

## Track C focus (Wei Ouyang, lead)

Track C is free exploration — no prescribed problem or tutorial. Participants form sub-teams of 2–3, pick any agentic problem relevant to their work, and build toward a live demo. The `docs/` site serves as the public-facing guide for participants.

The reusable KTH agent skills produced by Track C now live in their own
repository — **[github.com/kth-sci/kth-agent-skills](https://github.com/kth-sci/kth-agent-skills)**.
`Track_C/README.md` is a pointer to it.

## Working in this repo

- Event details, schedule, and contacts are in the PDF files at the root and on the `docs/` site.
- The `docs/` folder is served as a GitHub Pages site from the `track-b` branch.
- Track B and Track C each have their own `README.md` as the starting point for participants.
